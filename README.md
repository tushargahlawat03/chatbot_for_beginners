# chatbot_for_beginners
A simple menu-driven chatbot built using Python that responds to user-selected options in a continuous loop until exit.

This is a basic console-based chatbot developed in Python.
The chatbot interacts with users by displaying a menu of options and responding based on the selected input.

It is a beginner-friendly project designed to demonstrate:

Dictionary usage

Infinite loops (while True)

Conditional statements (if-elif-else)

User input handling

Basic chatbot logic
